# Legacy Vite frontend (Synthia workbench prototype)

This is the **current** Synthia web UI (Vite + React + TypeScript).

Per [ADR 0002](../docs/adr/0002-nextjs-not-now.md), we are **not** migrating to Next.js for v1.0. New product work continues here until a future replatform decision.

Build output is copied to `src/edagent_vivado/web/static/` for `edagent web` / `synthia web`.
